# User Data Pipeline Project

This project demonstrates a simple data pipeline built using Python.

The pipeline extracts user data from a public API, cleans and transforms the data using Pandas, stores structured tables in SQLite, and performs SQL analysis.

## Tools Used
- Python
- Pandas
- Requests
- SQLite

## Project Workflow
1. Fetch user data from API
2. Inspect and clean the dataset
3. Normalize the data into relational tables
4. Store tables in a SQLite database
5. Run SQL analysis queries to generate insights

## Files in this Repository

week4_project.py → Python script containing the full data pipeline  
users_cleaned.csv → Cleaned dataset exported from Pandas  
users_project.db → SQLite database with normalized tables  

## Example SQL Analysis

- Total users in dataset
- Users per city
- Users and their companies
- Combined view of users, cities, and companies
